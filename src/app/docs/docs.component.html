First steps!

The first step to use this api is to validate if there is data, we can do it from the content section, there you will find multiple sections that will allow you to create, edit, view and delete different data

Create edit and delete!

Each section will have a similar design, a button to create, a button to delete, a list with the created data, and in case you want to edit any data you just have to click on the desired record!

The home Api Consulter!

In the home page we can see the url of the api, a selector to choose between different sections and an id to obtain the selected record. Once you have entered that data, just press the magnifying glass icon and the data you want to see will appear!
